The open dataset

Every agency's latest grade and category scores in one file, refreshed daily, free to download and cite. The same scores behind the agency pages, published flat so a researcher or a state program can analyze the whole country at once.

Get the data

One row per agency: id, name, date, letter grade, overall score, the four category scores, and days until the service calendar expires. Built from the same dated artifacts the site runs on, so it is reproducible and versioned.

NTD certification readiness, by state

Since Report Year 2023, every National Transit Database reporter with fixed-route or deviated-fixed-route service has to publish a valid, current GTFS feed and certify it each year. This is how many of the tracked agencies look ready, nationally and per state. Readiness mirrors three pillars (published, valid, current); it is a heads-up, not an official determination.

How to use and cite it

Free to use, share, and build on. If you cite it, please link gtfsscorecard.org and note the date you pulled the data, since it refreshes daily. The scoring method, with citations, is in docs/rubric.md, how it maps to the standards is in docs/crosswalk.md, and the machine-readable weights are at scoring.json.

This is a data-quality lens, not an official compliance determination. The grade is computed on top of the canonical MobilityData GTFS validator.
