Every route, one map.

The route shapes of every tracked agency, drawn from each feed's own GTFS and combined on a single map. 8455 routes from 462 agencies are on it. Recolour by route type or by the agency's data-quality grade, and select a line for the agency and a link to its scorecard.

This map is a visual extra, not the accessible way to read the data. A map of this many route lines can't be a data table. For a screen-reader and keyboard friendly view, use the sortable agencies list; each agency's own scorecard carries a route-by-route table of its lines.
